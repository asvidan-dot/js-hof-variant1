# Практична робота №8: Data Transformation Library
**Варіант 1**

### Опис
Бібліотека функцій вищого порядку для трансформації даних, побудована на принципах функціонального програмування.

### Реалізовані функції:
* **map, filter, reduce** — власні реалізації базових методів масивів.
* **pipe / compose** — інструменти для композиції функцій.
* **curry** — каррінг функцій для часткового застосування.
* **memoize** — оптимізація обчислень через кешування результатів.
